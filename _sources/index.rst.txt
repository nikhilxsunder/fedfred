fedfred
=======

.. image:: _static/fedfred-logo.png
   :width: 300px
   :alt: FedFred Logo
   :class: only-light

A feature-rich Python package for interacting with the Federal Reserve Bank of St. Louis Economic Database (FRED).

Features
--------

* Pandas/Polars DataFrames
* Async support
* Local caching
* Rate limiting
* Type-safe API
